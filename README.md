# Portal de Datos Abiertos Republica Dominicana (datos-gob-do)

Portal de Datos Abiertos Republica Dominicana is a national government open-data portal for Dominican Republic running **CKAN**, exposing a standardized catalog API over ~1,054 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/datos-gob-do/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=datos-gob-do-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.11.3

## API
- **Portal de Datos Abiertos Republica Dominicana CKAN Action API** — base `https://datos.gob.do/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://datos.gob.do)
- Live endpoint verified: `https://datos.gob.do/api/3/action/status_show` · ~1,054 datasets
